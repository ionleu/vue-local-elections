# Vue local elections
A Vue.js simple application that displays local elections result in real time. See demo [hire](https://elections.lebo.md).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```
